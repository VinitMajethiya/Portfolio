# Vinit Atul Majethiya — Portfolio

> Personal portfolio website. Built from scratch. No templates.

## What This Is

A single-page portfolio designed to show who I am — not just what I can build. Built with vanilla HTML, CSS, and JavaScript. Dark, warm, editorial aesthetic. Every design decision documented in `docs/` (or root `.md` files).

## Stack

- HTML5 · CSS3 (custom properties, grids, transitions, keyframes) · Vanilla JS
- Fonts: Playfair Display · DM Mono · Cormorant Garamond (Google Fonts)
- Deployed on: Vercel / GitHub Pages

## Docs

The project details are defined in:

| File | Purpose |
|---|---|
| `DESIGN.md` | Color system, typography, components, animation catalogue |
| `CONTENT.md` | Copywriting guide and all section content |
| `STRUCTURE.md` | File structure, HTML skeleton, build order |
| `CHECKLIST.md` | Pre-launch quality gate |

## Running Locally

No build step needed.

```bash
git clone https://github.com/VinitMajethiya/VinitMajethiya.github.io.git
cd VinitMajethiya.github.io

# Option A — open directly
open index.html

# Option B — local server (recommended, avoids CORS issues)
npx serve .
# or
python3 -m http.server 3000
```
